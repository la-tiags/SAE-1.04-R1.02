Mission 2

Contexte

Cette mission porte sur l’exploitation de la base de données KDou via une interface low code (Access). L’objectif était d’explorer, manipuler et interroger la base de façon intuitive afin de faciliter le travail des utilisateurs non experts en SQL.

Indicateur intéressant : Produits américains à coût d’achat inférieur à 25 €

  • Objectif : Lister, par fournisseur, les produits américains présentant un coût d’achat inférieur à 25 €. Optimiser ainsi les approvisionnements à bas coût.
  • Interprétation : Cet indicateur aide KDou à identifier ses partenaires américains proposant des produits à forte marge potentielle. Il permet d’ajuster les commandes et de négocier des tarifs sur les volumes importants.
Exemple de résultats – Fournisseurs américains à coût réduit
Fournisseur Ville Produit Coût d’achat (€)
Exotic Liquids Burlington Chai 18,00
Specialty Meats Denver Pastrami 21,50
Code SQL utilisé (QBE/Access SQL): 
SELECT Fournisseur.Societe, Fournisseur.Ville, Produit.NomProd, Produit.CoutAchat
FROM Produit
RIGHT JOIN Fournisseur ON Produit.NoFour = Fournisseur.NoFour
WHERE Produit.CoutAchat < 25
  AND Fournisseur.Pays IN ("Etats-Unis","USA")
ORDER BY Produit.NomProd;
